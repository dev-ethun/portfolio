What is HTML ?

  • Html - Hyper Text Markup Language.
  • Use for Create Website Structure .
  • HTML consists of a series of elements .
  • HTML elements tell the browser how to display the content .

Example Explain -

  • এটি ডকুমেন্টের শুরুতে ব্যবহৃত হয়, যা ব্রাউজারকে HTML5 ভার্সন এর নির্দেশ দেয় ।


  • এটি হলো HTML ডকুমেন্টের মূল মূল ট্যাগ। যা ওয়েবপেজের সমস্ত কনটেন্ট এবং এলিমেন্ট ধারণ করে।


  • এটি HTML ডকুমেন্টের Meta তথ্য ধারণ করে যা ওয়েবপেজে সরাসরি দেখা যায় না। এতে title , CSS, এবং Script এর তথ্যও রাখা হয়।


  • ওয়েবপেজের শিরোনাম নির্ধারণ করে, যা ব্রাউজারের ট্যাবে দেখা যায়। এটি সার্চ ইঞ্জিনেও ওয়েবপেজের নাম হিসেবে দেখানো হয়।


  • এটি ওয়েবপেজের দৃশ্যমান অংশ ধারণ করে। এর ভেতরে টেক্সট, ছবি, ভিডিও, লিংক, বাটন ইত্যাদি থাকে যা ইউজার দেখতে ও ব্যবহার করতে পারে।


  • HTML Document এ বড় কোনো শিরোনাম দেখাতে এটি ব্যবহৃত হয় ।


  • HTML Document এ প্যারাগ্রাফ বা অনুচ্ছেদ লেখার জন্য এটি ব্যবহৃত হয়। এটি সাধারণ টেক্সট কনটেন্ট সাজানোর জন্য ব্যবহৃত হয়।।


What is HTML Element ?

HTML এলিমেন্ট হলো একটি ওয়েবপেজের নির্দিষ্ট অংশ বা উপাদান, যা সাধারণত একটি Start / Opening ট্যাগ, Content , এবং একটি End / Closing ট্যাগ নিয়ে গঠিত।

Note : Some HTML elements have no content (like the <br> , <hr> , <img> element). These elements are called empty elements. Empty elements do not have an end tag !
